# 環境
`iTerm2 3.5.13`

# 解決策
* `iTerm2 > Preferences > Profiles > Keys> Key Bindings`

1. `Presets`からSend Hex Code を選択
2. Keyboard Shortcutを押してバックスラッシュを入力
3. 文字コード入力欄に`5c`を入れる
